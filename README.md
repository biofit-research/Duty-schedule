# 당직 스케줄러 (Duty Scheduler)

구성원들의 당직을 공평하게 자동 분배하고 엑셀로 내보내는 단일 HTML 웹앱입니다. 별도 설치나 서버 없이 브라우저에서 바로 동작합니다.

## 기능

- 구성원 추가 / 삭제
- 구성원별 휴가일 지정 (해당 날짜는 당직에서 자동 제외)
- 평일 / 주말 각각 하루 배정 인원수 설정
- 매일 당직 횟수가 가장 적은 사람을 우선 배정해 공평하게 분배
- 같은 사람이 연속으로 당직 서지 않도록 방지 (옵션, 켜고 끌 수 있음)
- 인원이 부족하거나 규칙을 완전히 지킬 수 없을 때 경고 표시
- 결과를 엑셀(.xlsx) 파일로 내보내기 (당직표 시트 + 사람별 총 당직횟수 요약 시트)

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 구성원을 추가하고 각자의 휴가일을 입력합니다.
3. 연/월, 평일·주말 인원수, 연속 방지 여부를 설정합니다.
4. **당직표 생성** 버튼을 누릅니다.
5. **엑셀 파일로 내보내기** 버튼으로 .xlsx를 다운로드합니다.

## 분배 규칙

- 매일 정해진 인원(평일·주말 별도)을 배정합니다.
- 그 시점까지 당직 횟수가 가장 적은 사람을 우선 배정합니다.
- 휴가일에는 해당 인원을 배정하지 않습니다.
- 연속 방지 옵션이 켜져 있으면 전날 당직자는 후보에서 제외하되, 그렇게 하면 인원이 부족할 때만 예외적으로 배정합니다.

## 기술

- 순수 HTML / CSS / JavaScript (프레임워크 없음)
- 엑셀 내보내기: [SheetJS (xlsx)](https://sheetjs.com) — CDN으로 로드
- 다크/라이트 모드 친화적인 색상
