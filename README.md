# 당직 스케줄러 (Duty Scheduler)

구성원들의 당직을 공평하게 자동 분배하고 엑셀로 내보내는 단일 HTML 웹앱입니다. 별도 설치나 서버 없이 브라우저에서 바로 동작합니다.

## 기능

- 구성원 추가 / 삭제
- 구성원별 휴가일 지정 (해당 날짜는 당직에서 자동 제외)
- **구성원별로 평일 / 주말 당직 일수를 직접 지정**
- 지정한 일수만큼 한 달에 걸쳐 고르게 분산 배정
- 같은 사람이 연속으로 당직 서지 않도록 방지 (옵션, 켜고 끌 수 있음)
- 할당을 다 채우지 못하거나 합계가 맞지 않을 때 경고 표시
- 결과를 엑셀(.xlsx) 파일로 내보내기 (당직표 시트 + 사람별 평일·주말·총 당직횟수 요약 시트)

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 구성원을 추가하고 각자의 휴가일, 평일 당직 수, 주말 당직 수를 입력합니다.
3. 연/월과 연속 방지 여부를 설정합니다.
4. **당직표 생성** 버튼을 누릅니다.
5. **엑셀 파일로 내보내기** 버튼으로 .xlsx를 다운로드합니다.

## 분배 규칙

- 구성원 각자에게 정한 평일·주말 당직 일수만큼 배정합니다.
- 배정은 그 달 전체에 최대한 고르게 분산됩니다.
- 휴가일에는 해당 인원을 배정하지 않습니다.
- 연속 방지 옵션이 켜져 있으면 전날 당직자는 다음 날 후보에서 제외하되, 그래야만 채울 수 없을 때는 예외적으로 배정합니다.
- 평일 당직 합계는 그 달의 평일 수, 주말 당직 합계는 주말 수와 맞아야 빈자리 없이 하루 1명씩 채워집니다. 맞지 않으면 빈 날이 생기거나 하루 2명 이상 배정되며 경고가 표시됩니다.

## 기술

- 순수 HTML / CSS / JavaScript (프레임워크 없음)
- 엑셀 내보내기: [SheetJS (xlsx)](https://sheetjs.com) — CDN으로 로드
- 다크/라이트 모드 친화적인 색상
